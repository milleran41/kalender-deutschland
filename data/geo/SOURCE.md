# Map Data Source

The Germany and federal-state GeoJSON files in this folder are kept as local
extension assets so the atlas works offline.

Open GitHub sources used for this atlas:

- `m-ad/geofeatures-ags-germany` - outlines for German counties and states in
  GeoJSON/TopoJSON format.
- `isellsoap/deutschlandGeoJSON` - administrative borders of Germany in
  GeoJSON format with multiple quality levels.

The atlas uses local copies and shows an attribution note inside the map UI.
